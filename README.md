The project develops a low-cost heart rate and blood oxygen saturation (SpO₂) monitoring system using the MAX30102 optical sensor integrated with an ESP32 microcontroller. The sensor captures optical signals from the fingertip via red (660nm) and infrared (880nm) LEDs. The ESP32 processes the signals using a Peak Detection algorithm for heart rate calculation and a RED/IR ratio method for SpO₂ estimation. Results are displayed in real time on an OLED screen and transmitted to ThingSpeak and Blynk IoT platforms via WiFi. The system also incorporates RGB LEDs and a buzzer to alert users when measurements exceed safe thresholds.

Đồ án xây dựng hệ thống đo nhịp tim và nồng độ oxy trong máu (SpO₂) chi phí thấp sử dụng cảm biến quang học MAX30102 kết hợp với vi điều khiển ESP32. Cảm biến thu tín hiệu quang học từ ngón tay thông qua hai LED đỏ (660nm) và hồng ngoại (880nm), ESP32 xử lý tín hiệu bằng thuật toán phát hiện đỉnh (Peak Detection) để tính nhịp tim và phương pháp tỉ lệ RED/IR để tính SpO₂. Kết quả được hiển thị theo thời gian thực trên màn hình OLED, đồng thời truyền lên nền tảng IoT ThingSpeak và Blynk qua WiFi. Hệ thống còn tích hợp LED RGB và buzzer để cảnh báo khi chỉ số vượt ngưỡng an toàn.
